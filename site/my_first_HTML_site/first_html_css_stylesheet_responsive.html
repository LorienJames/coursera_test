CSS Anatomy and Selectors

The anatomy of a CSS rule can be defined by 'secector {property: value;}' with the content in the {} being the 'declaration'. A CSS rule can contain multiple declaration and can range from zero to many. A collection of CSS rules is called a style sheet and it is best practice for this to be a seperate file linked with <link rel="stylesheet" href="style.css">. The CSS code can be placed in the HTML document itself with <style> or placed within an element style=.

CSS has three different selectors: element secector, class selector, id selector. The element selector targets an element p {color: blue;} with p being the target element for <p>. Whereas class selector you can set a class to target .blue {color: blue;} and then set a class within the element <p class="blue">. Id selectors will target id set within elements and #section1 {color: blue;} will target id="section1" within an element. You can specify multiple selectors by placing a ',' following the selector preceeding another selector. Selectors can be combined to consider elements wihin a class, child of another element or a decendant. Examples of this are:

  1. p.big {font-size: 20px;} will target any <p> with class="big"
  2. article > p {color: blue;} will target any <p> directly within an <article>
  3. article p {color: blue;} will target any <p> that is a descendant of an <article>

There are also pseudo-class selectors and these are defined by :selector or for example a:link or a:visited. There are many of these selectors and we note the following:

  1. :link
  2. :visited
  3. :hover
  4. :active
  5. :nth-child(...) for example section4 li:nth-child(odd) {background-color: #00bfff;}

CSS Box Model and Floating

A function introduced in CSS3 is border box and sets the box size to include padding and border. This can be specified in a styling sheet as *{box-sizing: border-box}. Note we use a star selector to target all elements as border box is not inherited by child elements.

CSS Responsive Design

Media queries adjust the look of a website according to screen size. An example: @media (min-width: 1200) {p {color: blue;}}

This can be viewed as: media feature(resolves to true or false) and then the style within the feature is either applied or not.

    Common features include:
  • @media (max-width) {...}
  • @media (min-width) {...}
  • @media (orientation: portrait) {...}
  • @media screen {...}
  • @media print {...}

Media features can be combined using logic operators: and , (or).
