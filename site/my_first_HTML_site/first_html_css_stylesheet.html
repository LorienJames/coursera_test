This is writen using the character set utf-8. I'm using the tags <body>, <h1>, </h1>, <p>, </p>, <div>, </div> to open and close the body, heading, paragraph and division. The < and > sybols have been escaped using the character entity references & l t ; and & g t ;. The character entity refence & n b s p ; has also been used to create the spaces between the three words (character entity references) in the previouse sentence to ensure that all three words are wrapped on the same line.

Lists

An Unordered List using the following tags:
  • <ul> to open the unordered list
    • <li> to open a list item
    • </li> to close a list item
  • </ul> to close the unordered list
An Ordered List using the tags:
  1. <ol> to open an ordered list
    1. <li> to open a list item
    2. </li> to close a list item
  2. </ol> to close the ordered list
